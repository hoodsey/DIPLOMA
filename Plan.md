# Описание валидных данных для тестов
Генерируем библиотекой faker. 
Общее описание валидных данных форм:
>1. Номер карты: Банковская карта из 16 цифр, начиная не с 0.
>  
> 2. месяц: 2 цифры с текушего года и текущего месяца до текущего месяца 5+ лет
>
> 3. год: две цифры, с текушего года и текущего месяца до текущего месяца +5 лет
>
>4.  CVV-код: 3 цифры исключая три 000
 
Пример: 
> номер карты: 4444 4444 4444 4441
>
> 
> месяц: 08
>
> год: 26
>
> владелец: IVAN IVANOV
>
> CVC|CVV: 111
>

## ## Интервальная оценка
--------------
Написание сценариев + написание автотестов(включая настройку среды)+ проверка автотестов+  риски 

> 6ч+18ч+6ч+6ч = 36ч = примерно 1 неделя

# План сдачи работ: когда будут готовы автотесты

Исходя из оценки 1 неделя без учета занятости. C учетом занятости на работи итоговый срок - 2 недели
